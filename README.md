# EDII-Parcial2

# 🛒 Gestión de Catálogo de Productos - Algoritmos de Ordenamiento y Búsqueda

Proyecto desarrollado para el **Parcial 2** de la asignatura de **Estructuras de Datos II**. Implementa y compara algoritmos de ordenamiento y búsqueda aplicados a un catálogo realista de productos para una tienda en línea.

## 📋 Descripción
El sistema simula un módulo de backend para una tienda online que requiere:
- Generación de un catálogo de 50 productos con datos coherentes y aleatorios.
- Ordenamiento de productos por **precio (ascendente)** y **calificación promedio (descendente)** usando tres algoritmos distintos.
- Búsqueda eficiente de productos por **ID** (Búsqueda Binaria) y por **nombre/subcadena** (Búsqueda Lineal).
- Medición precisa de tiempos de ejecución y análisis de rendimiento basado en complejidad algorítmica.

## 🛠️ Requisitos
- Python 3.8 o superior
- **Sin dependencias externas** (solo se utilizan librerías estándar: `random`, `time`, `dataclasses`, `typing`)
